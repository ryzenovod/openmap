# Medical Geo Analytics Platform

Новый backend-first проект внутренней медицинской геоаналитической системы.

## Текущий scope
- FastAPI backend
- PostgreSQL + PostGIS
- staging / core / mart
- CSV import MVP
- aggregate API для карты и графиков

## Legacy policy
Legacy используется только как источник:
- словарей
- field mapping
- import semantics
